## Why we will help you:

Or: What do we do? Although we have already drawn the lines of our future path, they have not yet been drawn. We plan to start as a dev.to based blogging organization, with different series running contemporaneously. Discord will allow us to be more user-focused. We don't want to be static. The main objective of going beyond a simple blog is to empower communication between developers and pair programming.
